Números ordinales

Del 1º al 10º cada número tiene un nombre propio. A partir del 11º se forma con el número cardinal quitando la última vocal y añadiendo "-esimo".
Número Italiano Español
1º primo primero

Números decimales

En italiano se usa la coma para números decimales matemáticos. Para las frecuencias de radio se usa punto.
