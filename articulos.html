Artículos Definidos e Indefinidos en Italiano

Artículos Definidos (il, lo, la, i, gli, le)

Masculino Singular

  • il - Se usa antes de consonantes (excepto s + consonante, z, ps, gn, y):
    Ejemplo: il libro (el libro), il cane (el perro)
  • lo - Se usa antes de:
    • s + consonante: lo studente (el estudiante)
    • z: lo zaino (la mochila)
    • ps: lo psicologo (el psicólogo)
    • gn: lo gnomo (el gnomo)

Femenino Singular

  • la - Se usa para todos los sustantivos femeninos:
    Ejemplo: la casa (la casa), la strada (la calle)

Masculino Plural

  • i - Plural de "il":
    Ejemplo: i libri (los libros)
  • gli - Plural de "lo":
    Ejemplo: gli studenti (los estudiantes)

Femenino Plural

  • le - Plural de "la":
    Ejemplo: le case (las casas)

Artículos Indefinidos (un, uno, una)

Masculino

  • un - Se usa antes de consonantes:
    Ejemplo: un libro (un libro)
  • uno - Se usa antes de:
    • s + consonante: uno studente (un estudiante)
    • z: uno zaino (una mochila)
    • ps: uno psicologo (un psicólogo)
    • gn: uno gnomo (un gnomo)

Femenino

  • una - Se usa para todos los sustantivos femeninos:
    Ejemplo: una casa (una casa)

Ejemplos Prácticos

  • Il ragazzo → I ragazzi (El chico → Los chicos)
  • Lo specchio → Gli specchi (El espejo → Los espejos)
  • La ragazza → Le ragazze (La chica → Las chicas)
  • Un amico (Un amigo)
  • Uno spazio (Un espacio)
  • Una amica (Una amiga)
